On top of the base theme provided by {\ttfamily doxygen-\/awesome.\+css}, this repository comes with Javascript extensions that require additional setup steps to get them running.

The extensions require customizations in the header HTML template. This is how you can create the default template with Doxygen\+:


\begin{DoxyEnumerate}
\item Create default header template\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{doxygen\ -\/w\ html\ header.html\ delete\_me.html\ delete\_me.css}

\end{DoxyCode}

\item Reference the template in your {\ttfamily Doxyfile}\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{HTML\_HEADER\ \ \ \ \ \ \ \ \ \ \ \ =\ header.html}

\end{DoxyCode}

\end{DoxyEnumerate}

\hypertarget{md_doxygen-awesome-css_2docs_2extensions_extension-page-navigation}{}\doxysection{\texorpdfstring{Page Navigation}{Page Navigation}}\label{md_doxygen-awesome-css_2docs_2extensions_extension-page-navigation}
\begin{DoxyWarning}{Warning}
Experimental feature! Please report bugs \href{https://github.com/jothepro/doxygen-awesome-css/issues}{\texttt{ here}}.
\end{DoxyWarning}
To allow the user to easily navigate from one document to another, "{}\+Next"{} and "{}\+Previous"{} buttons can be added at the end of a Markdown document.\hypertarget{md_doxygen-awesome-css_2docs_2extensions_autotoc_md17}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{md_doxygen-awesome-css_2docs_2extensions_autotoc_md17}
The feature is shipped inside the default {\ttfamily doxygen-\/awesome.\+css}. No additional stylesheets or scripts need to be added.\hypertarget{md_doxygen-awesome-css_2docs_2extensions_autotoc_md18}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{md_doxygen-awesome-css_2docs_2extensions_autotoc_md18}
The following conditions must be met for the feature to work properly\+:
\begin{DoxyItemize}
\item The navigation must be inside a Markdown table with 1-\/2 columns.
\item The alignment of the column defines the alignment of the arrow on the navigation button.
\item the table must be wrapped inside a {\ttfamily \texorpdfstring{$<$}{<}div\texorpdfstring{$>$}{>}} with the class {\ttfamily section\+\_\+buttons}.
\end{DoxyItemize}
