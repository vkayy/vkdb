{\bfseries{Doxygen Awesome}} is a custom CSS theme for Doxygen HTML documentation with lots of customization parameters.\hypertarget{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md30}{}\doxysection{\texorpdfstring{Motivation}{Motivation}}\label{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md30}
I really like how the Doxygen HTML documentation is structured! But IMHO it looks a bit outdated.

To use the theme when generating your documentation, bring the required CSS and JS files from this repository into your project.

This can be done in several ways\+:


\begin{DoxyItemize}
\item manually copying the files
\item adding the project as a Git submodule
\item downloading the project with CMake Fetch\+Content
\item adding the project as a npm/xpm dependency
\item installing the theme system-\/wide
\end{DoxyItemize}

All theme files are located in the root of this repository and start with the prefix {\ttfamily doxygen-\/awesome-\/}. You may not need all of them. Follow the install instructions to figure out what files are required for your setup.\hypertarget{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md34}{}\doxysubsection{\texorpdfstring{Git submodule}{Git submodule}}\label{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md34}
For projects that use git, add the repository as a submodule and check out the desired release\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{git\ submodule\ add\ https://github.com/jothepro/doxygen-\/awesome-\/css.git}
\DoxyCodeLine{cd\ doxygen-\/awesome-\/css}
\DoxyCodeLine{git\ checkout\ v2.3.4}

\end{DoxyCode}
\hypertarget{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md35}{}\doxysubsection{\texorpdfstring{CMake with Fetch\+Content}{CMake with Fetch\+Content}}\label{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md35}
For project that build with CMake, the {\ttfamily Fetch\+Content} module can be used to download the repository at configure-\/time.

Add the following snippet to your {\ttfamily CMake\+Lists.\+txt}


\begin{DoxyCode}{0}
\DoxyCodeLine{include(FetchContent)}
\DoxyCodeLine{FetchContent\_Declare(}
\DoxyCodeLine{\ \ \ \ doxygen-\/awesome-\/css}
\DoxyCodeLine{\ \ \ \ URL\ https://github.com/jothepro/doxygen-\/awesome-\/css/archive/refs/heads/main.zip}
\DoxyCodeLine{)}
\DoxyCodeLine{FetchContent\_MakeAvailable(doxygen-\/awesome-\/css)}
\DoxyCodeLine{}
\DoxyCodeLine{\#\ Save\ the\ location\ the\ files\ were\ cloned\ into}
\DoxyCodeLine{\#\ This\ allows\ us\ to\ get\ the\ path\ to\ doxygen-\/awesome.css}
\DoxyCodeLine{FetchContent\_GetProperties(doxygen-\/awesome-\/css\ SOURCE\_DIR\ AWESOME\_CSS\_DIR)}
\DoxyCodeLine{}
\DoxyCodeLine{\#\ Generate\ the\ Doxyfile}
\DoxyCodeLine{set(DOXYFILE\_IN\ \$\{CMAKE\_CURRENT\_SOURCE\_DIR\}/doc/Doxyfile.in)}
\DoxyCodeLine{set(DOXYFILE\_OUT\ \$\{CMAKE\_CURRENT\_BINARY\_DIR\}/Doxyfile)}
\DoxyCodeLine{configure\_file(\$\{DOXYFILE\_IN\}\ \$\{DOXYFILE\_OUT\}\ @ONLY)}

\end{DoxyCode}


This downloads the latest main (but any other revision could be used) and unpacks in the build folder. The {\ttfamily Doxyfile.\+in} can reference this location in the {\ttfamily HTML\+\_\+\+EXTRA\+\_\+\+STYLESHEET} field


\begin{DoxyCode}{0}
\DoxyCodeLine{HTML\_EXTRA\_STYLESHEET\ \ =\ @AWESOME\_CSS\_DIR@/doxygen-\/awesome.css}

\end{DoxyCode}


When the configure stage of CMake is run, the {\ttfamily Doxyfile.\+in} is rendered to Doxyfile and Doxygen can be run as usual.\hypertarget{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md36}{}\doxysubsection{\texorpdfstring{npm/xpm dependency}{npm/xpm dependency}}\label{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md36}
In the npm ecosystem, this project can be added as a development dependency to your project\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{cd\ your-\/project}
\DoxyCodeLine{npm\ install\ https://github.com/jothepro/doxygen-\/awesome-\/css\#v2.3.4\ -\/-\/save-\/dev}
\DoxyCodeLine{}
\DoxyCodeLine{ls\ -\/l\ node\_module/@jothepro/doxygen-\/awesome-\/css}

\end{DoxyCode}


Similarly, in the \href{https://xpack.github.io}{\texttt{ x\+Pack}} ecosystem, this project can be added as a development dependency to an \href{https://xpack.github.io/xpm/}{\texttt{ {\ttfamily xpm}}} managed project.\hypertarget{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md37}{}\doxysubsection{\texorpdfstring{System-\/wide}{System-\/wide}}\label{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md37}
You can even install the theme system-\/wide by running {\ttfamily make install}. The files will be installed to {\ttfamily /usr/local/share/} by default, but you can customize the install location with {\ttfamily make PREFIX=/my/custom/path install}.\hypertarget{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md38}{}\doxysubsection{\texorpdfstring{Choosing a layout}{Choosing a layout}}\label{md_doxygen-awesome-css_2_r_e_a_d_m_e_autotoc_md38}
There are two layout options. Choose one of them and configure Doxygen accordingly\+:




\begin{DoxyItemize}
\item {\bfseries{Base Theme}}   Comes with the typical Doxygen titlebar. Optionally the treeview in the sidebar can be enabled.

Required files\+: {\ttfamily doxygen-\/awesome.\+css}

Required {\ttfamily Doxyfile} configuration\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{GENERATE\_TREEVIEW\ \ \ \ \ \ =\ YES\ \#\ optional.\ Also\ works\ without\ treeview}
\DoxyCodeLine{DISABLE\_INDEX\ =\ NO}
\DoxyCodeLine{FULL\_SIDEBAR\ =\ NO}
\DoxyCodeLine{HTML\_EXTRA\_STYLESHEET\ \ =\ doxygen-\/awesome-\/css/doxygen-\/awesome.css}
\DoxyCodeLine{HTML\_COLORSTYLE\ \ \ \ \ \ \ \ =\ LIGHT\ \#\ required\ with\ Doxygen\ >=\ 1.9.5}

\end{DoxyCode}

\item {\bfseries{Sidebar-\/\+Only Theme}}   Hides the top titlebar to give more space to the content. The treeview must be enabled in order for this theme to work.

Required files\+: {\ttfamily doxygen-\/awesome.\+css}, {\ttfamily doxygen-\/awesome-\/sidebar-\/only.\+css}

Required {\ttfamily Doxyfile} configuration\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{GENERATE\_TREEVIEW\ \ \ \ \ \ =\ YES\ \#\ required!}
\DoxyCodeLine{DISABLE\_INDEX\ \ \ \ \ \ \ \ \ \ =\ NO}
\DoxyCodeLine{FULL\_SIDEBAR\ \ \ \ \ \ \ \ \ \ \ =\ NO}
\DoxyCodeLine{HTML\_EXTRA\_STYLESHEET\ \ =\ doxygen-\/awesome-\/css/doxygen-\/awesome.css\ \(\backslash\)}
\DoxyCodeLine{\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ doxygen-\/awesome-\/css/doxygen-\/awesome-\/sidebar-\/only.css}
\DoxyCodeLine{HTML\_COLORSTYLE\ \ \ \ \ \ \ \ =\ LIGHT\ \#\ required\ with\ Doxygen\ >=\ 1.9.5}

\end{DoxyCode}

\end{DoxyItemize}

\begin{DoxyWarning}{Warning}

\begin{DoxyItemize}
\item This theme is not compatible with the {\ttfamily FULL\+\_\+\+SIDEBAR = YES} option provided by Doxygen!
\item {\ttfamily HTML\+\_\+\+COLORSTYLE} must be set to {\ttfamily LIGHT} since Doxygen 1.\+9.\+5!
\end{DoxyItemize}
\end{DoxyWarning}
